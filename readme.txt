
	Клиент-серверное приложение для передачи информации о системе

1. запуск сервера: ./server 5 1122 tcp
2. запуск клиента: ./client 127.0.0.1 1122 tcp

*. run_tcp.sh(run_udp.sh) - запуск нескольких клиентов разом(для проверки многопоточности)

**. файл tmp.txt обязательно должен находиться в папке на момент запуска приложения
